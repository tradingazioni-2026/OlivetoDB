# 🫒 OlivetoDB

Database scientifico sulla compatibilità di impollinazione tra cultivar di olivo.

## Funzionalità

- **Database** – 53 cultivar con gruppo SI (G1/G2), autofertilità, fioritura, resa, impollinatori consigliati
- **Selettore Oliveto** – inserisci le tue piante e ricevi consigli su cosa aggiungere
- **Verifica Compatibilità** – controlla se due cultivar si impollinano tra loro
- **Dashboard** – panoramica del sistema DSI e dati sull'impollinazione assistita

## Come pubblicare su GitHub Pages

1. Crea un repository su GitHub (es. `oliveto-db`)
2. Carica il file `index.html`
3. Vai su **Settings → Pages → Source: main branch / root**
4. Il sito sarà disponibile su `https://tuonome.github.io/oliveto-db`
